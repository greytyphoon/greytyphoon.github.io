Pathfinder 2nd-edition Houserules

Common Rules

  • XP will be shared across all players. Everybody will always have exactly the same XP.
  • All characters gain a Free Archetype.
  • All characters have Automatic Bonus Progression.
  • All characters gain one extra skill increase at 3rd, 7th, 15th level.
  • Exploration actions will cause tension dice to be added to the pool, for each 10 minutes.

Rarity

When using elements that are not [Common], you should check with the GM.

  • [Uncommon] elements are generally allowed. Please use them with some moderation.
  • [Rare] elements are generally restricted to one per character.
  • [Unique] elements generally should be obtained in-game.

Allowed Material

  • Official Paizo content is always allowed.
  • Third-party content and homebrews are generally allowed, but you should check with the GM.

Optional Rules - Not used unless specifically mentioned

  1. Dual-Class PCs Gestalt

    A classic very-high-power game. Most likely used at low player counts.
    Note that because Dual-Class characters already get two class feats every even level, this rule generally replaces Free Archetype.

  2. Pushed Rolls from Call of Cthulhu

    Outside of combat, when you fail at a roll, you can decide to push your luck. You must describe another way that you retry, giving it all you have. This allows you to reroll the initial failed roll. However, should you fail again, you automatically get a critical failure. You cannot push a roll again until another player pushes a roll.

  3. Connections from Traveller RPG

    A character can become Trained in an extra skill at 1st level. It must be a skill that is related to, or used in, a shared history that this character has with another player character in the same game. Work with other players to create those stories, which happened off-screen, usually before the start of the campaign.
